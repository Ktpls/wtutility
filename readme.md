#*<用户手册>*
本文为用于WT的带火控光学防空雷达的使用说明
#界面介绍
内蓝绿，中白，外红的圆圈是搜索圈
绿色细圆圈是飞机位置
蓝色直线段是飞机速度
红色细圆圈是火控预喵圈
#用法
用搜索圈套住飞机
按下F10开始锁定
按下F11取消锁定
按下F12关闭火控
#防空基础知识与使用注意事项
不要隔着十万八千里开火
太远也锁定不上
锁定上了预喵圈也不准确
不要按住左键不松手
套着圈也不总能打中，要用圈去撞枪线
锁定时高速移动镜头容易丢失目标，要么慢一点，要么重新锁定
锁定有抗干扰能力，但干扰越少越好
尽量不要让飞机靠近干扰物，例如树，房子，山，黑烟，炮口火光和烟，其他飞机等
能治打不准，但救不了火花
#设置
**不懂能不能改，不懂怎么改，那就不要改**
##w
屏幕宽度，单位像素
##h
屏幕高度，单位像素
##FPS
开高了会卡
配置够开高FPS，则可以考虑提高stablamb
##stablamb
取值范围0-1
调低预喵圈对机动更灵敏
调高预喵圈更稳定
FPS足够高，则越高越好
##camerestablizersubsamplerate
取值范围0-1
调低速度快
调高预喵圈更稳定
##planetrackerchannel
'G','B'二选一
选'G'抗干扰能力强，速度慢
选'B'抗干扰能力弱，速度快
##searchrange
大于零
调低抗干扰好
调高高速移动镜头不容易丢失目标
##targetwingspan
目标翼展，单位千米
建议选一个比较通用的翼展
##vbullet
炮弹初速，单位千米每秒
建议选一个比较通用的初速
##innerhue
搜索环的色调
随便选一个好看的
